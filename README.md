# LLM 양자화 수준 발표 웹앱

정적 웹 프레젠테이션입니다. GitHub Pages에서 바로 동작합니다.

## 실행

1. 이 폴더의 `index.html`, `styles.css`, `app.js`를 GitHub 저장소 루트에 업로드합니다.
2. GitHub 저장소의 **Settings → Pages**에서 Branch를 `main` / root로 설정합니다.
3. 발급된 GitHub Pages URL로 접속합니다.

## 조작

- 다음/이전: `→`, `←`, Space
- 블랙아웃: `B`
- 발표자 노트: `P`
- 슬라이드 목록: `O`
- 전체 화면: 우상단 버튼
